ATRIBUTO DE JS O QUE ELE CONFIGURA
alert() Exibe uma caixa de diálogo de alerta com uma mensagem e um botão "OK".
console.log() Escreve uma mensagem no console de depuração do navegador.
document.getElementById() Retorna o elemento que tem o ID especificado.
document.querySelector() Retorna o primeiro elemento que corresponde a um seletor CSS especificado.
document.createElement() Cria um novo elemento HTML especificado pelo nome da tag.
addEventListener() Registra um evento a ser disparado quando uma ação específica ocorre no elemento.
setTimeout() Define um cronômetro para executar uma função ou instrução depois de um intervalo de tempo especificado.
setInterval() Chama uma função ou avalia uma expressão em intervalos de tempo especificados (repetidamente).
clearTimeout() Interrompe a execução de uma função previamente estabelecida com setTimeout.
clearInterval() Interrompe a execução repetitiva de uma função definida com setInterval.
Math.random() Retorna um número pseudo-aleatório entre 0 e 1.
Math.floor() Arredonda um número para baixo para o inteiro mais próximo.
Math.ceil() Arredonda um número para cima para o inteiro mais próximo.
Math.round() Arredonda um número para o inteiro mais próximo.
parseInt() Converte uma string para um número inteiro.
parseFloat() Converte uma string para um número de ponto flutuante (decimal).
isNaN() Verifica se um valor é "NaN" (não é um número).
typeof Retorna o tipo de uma variável ou expressão.
if...else Declara uma estrutura condicional. Executa um bloco de código se a condição for verdadeira, e outro bloco se for falsa.
for Cria um loop que é executado enquanto uma condição for verdadeira.
while Cria um loop que é executado enquanto a condição for verdadeira.
try...catch Trata erros que ocorrem em blocos de código, permitindo a execução de código alternativo.
throw Lança uma exceção (erro).
function Define uma função que pode ser chamada posteriormente.
return Retorna um valor de uma função.
var, let, const Declaram variáveis. "var" tem escopo de função, "let" e "const" têm escopo de bloco, com "const" para constantes.
JSON.parse() Converte uma string JSON em um objeto JavaScript.
JSON.stringify() Converte um objeto JavaScript em uma string JSON.
Array.push() Adiciona um ou mais elementos ao final de um array.
Array.pop() Remove o último elemento de um array.
Array.shift() Remove o primeiro elemento de um array.
Array.unshift() Adiciona um ou mais elementos ao início de um array.
Array.map() Cria um novo array com os resultados da chamada de uma função para cada elemento no array original.
Array.filter() Cria um novo array com todos os elementos que passam no teste implementado pela função fornecida.
Array.reduce() Aplica uma função a cada elemento do array (da esquerda para a direita) para reduzir o array a um único valor.
Array.forEach() Executa uma função para cada elemento de um array.
